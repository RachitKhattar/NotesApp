# NotesApp
Assignment Project for wander.today

# Description
A basic iOS notes app, that fetches data from a Stubbed Request and has a Add Note functionality.<br/>
Storing all the notes in User Defaults.<br/>
All basic functionality can be found in Apps Default Notes.
